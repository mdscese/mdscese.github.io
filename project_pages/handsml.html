Hand Detection Model
❯

Description:

A currently in the works machine learning model that tracks the users hand in real time, with the intent to classify the shape of the hand into the sign language alphabet.

Implementation:

This machine learning model project is written in Python and employs various libraries for image processing and deep learning tasks. The project utilizes OpenCV for capturing image data, PyTorch for training the models, and CUDA for GPU acceleration. The primary objective of the project is to retrain the Faster R-CNN model using the Ego Hands dataset for hand detection and subsequently use PyTorch to develop a regular Convolutional Neural Network (CNN) for hand shape classification.

Technologies:

Python, PyTorch, OpenCV, CUDA
