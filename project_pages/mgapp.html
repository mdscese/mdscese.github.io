Multiplayer Game App
❯

Description:

A team project developed in early 2020. It allows for android phones to connect to each other through an assortment of connection types and facilitates the ability to play small games with each other. These games include tic-tac-toe, chess, and hangman to name a few.

Implementation:

We developed a small game-playing app in Java utilizing the toolkit provided by Android Studio. By integrating Google Nearby, a now deprecated API, the app enables connectivity between Android phones through various mediums such as Bluetooth, WiFi, and Data. This functionality as well as our developed game packet systemn allows users to play the game with nearby individuals. But to ensure that the games were still playable even when fully disconnected we developed smart artifical intelligences for each game.

Technologies:

Java, Android Studio, Google Nearby
