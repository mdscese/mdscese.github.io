Infinite Cities Game
❯

Description:

A game currently in development, aiming to provide players with an immersive experience of exploring an endless urban landscape. While allowing me to explore creative design as well as interesting game mechanics.

Implementation:

This game is exclusively built using Godot, an open-source game engine developed as a competitor to engines like Unity. Internal scripts heavily prioritize randomized content generation, relying extensively on wave form collapse techniques to construct diverse environments dynamically. Additionally, all assets are personally crafted by me using Blender, a powerful 3D modeling software.

Technologies:

GDScript, Godot, Blender
