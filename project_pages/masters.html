Masters Report
❯

Description:

Published in the fall of 2023, this paper introduces the GraphQL Mesh federated API (Application Programming Interface) gateway project, a comprehensive initiative implemented using GraphQL Mesh to solve data related issues within the USW-DSS (Undersea Warfare - Decision Support System). The project contributes to the evolving discourse on the pivotal role of Data Fabrics and Data Meshes in dismantling the barriers imposed by digital data silos.
